# Project Memory

## Core
Healthtech app for clinical tracking of complex wounds (pressure ulcers, diabetic foot, surgical).
Brand: **CuraTrack**. Logo at `@/assets/curatrack-logo.png` (blue cross + green checkmark swoosh).
Palette: Primary blue `hsl(217 89% 38%)` + brand green `hsl(142 71% 45%)`. Gradient blue→green for hero/brand accents. Montserrat headings, Open Sans body.

## Memories
- [Branding Guidelines](mem://design/branding) — CuraTrack identity, blue/green palette, typography
- [Clinical Tracking](mem://features/clinical-tracking) — Core domain models for Patients, Cases (Wounds), and Evolutions
- [Dashboard & Scheduling](mem://features/scheduling) — Metrics, agenda, next control dates, and calendar color-coding
- [PDF Export](mem://features/pdf-export) — Export capabilities for clinical histories and reports
- [Media Management](mem://features/media-management) — Clinical photography capture, gallery, and timeline
- [Multi-user Demo](mem://features/multi-user-demo) — localStorage auth, 3 demo accounts, team + per-patient shares with roles
